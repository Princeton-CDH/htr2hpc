Click here for information on pretrained_ models.

All users of this beta test should have access to a series of models with pretrained_ at the start of their names, covering several languages and scripts. The sources of these models are detailed below. Please refer to the citations there when citing use of these models in your work.

Model Name Language(s) Script(s) Sources
pretrained_french French Latin
Click for model citation. The model file comes from the following source:
  • Gabay, S., Chagué, A., & Clérice, T. (2024). HTR-United - Manu Mc Fondue (Manuscripts of Modern and Contemporaneous French - Manu McFrench v4) (Version v4). Zenodo. https://doi.org/10.5281/zenodo.10886224.
pretrained_german German Latin
Click for model citation. The model file comes from the following source:
  • Weil, S. (2023). HTR model for German manuscripts trained from several datasets. Zenodo. https://doi.org/10.5281/zenodo.7933463.
pretrained_greek Greek Greek (minuscule)
Click for model citation. The model file comes from the following source:
  • Guénette, M., Verstraete, M., Chagué, A., & Vitali-Rosati, M. (2024). HTR Model Palatinus graecus 23 (Meleagre-NFC). Zenodo. https://doi.org/10.5281/zenodo.10932711.
pretrained_hebrew Hebrew Hebrew (Ashkenazi, Italian, and Sephardi)
Click for model citation. The model file comes from the following source:
  • Daniel STOEKL BEN EZRA. (2021). Medieval Hebrew manuscripts version 1.0. Zenodo. https://doi.org/10.5281/zenodo.5468286.


To find further models, one starting point is the OCR/HTR model repository on Zenodo. To find further transcription datasets, see the HTR-United catalogue of published training data for a starting point.
